# Lab 01 – Create a Microsoft Purview DLP Policy

## Objective

Create and test a Microsoft Purview Data Loss Prevention policy.

## Environment

- Microsoft 365 E5 Demo Tenant
- Microsoft Purview Portal

## Steps Performed

1. Opened Microsoft Purview.
2. Navigated to Data Loss Prevention.
3. Created a new DLP policy.
4. Selected a template.
5. Configured locations.
6. Configured rules.
7. Enabled the policy.
8. Tested with sample files.

## Result

The policy successfully detected sensitive information and generated DLP alerts.

## Lessons Learned

- How DLP policies are evaluated.
- How Sensitive Information Types are detected.
- How alerts appear in Purview.
